how add the comments in the header of the ftpcommand file

You should add the “!” to the begin of the comment line. And you should add a space character between !@rem and the comment content.

Conclusions

Linux Unix Os: use the !# to the begin of the comment. Windows: use the !@rem to the begin of the comment. what is “!”

!go to shell execute this command, so we can use the shell or dos comments syntax. @ tell the shell don’t echo in this command.
